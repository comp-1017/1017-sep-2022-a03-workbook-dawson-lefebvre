Add extension live server fir GO Live execution within VSC

  1. Open VSC and select your extension icon
  2. Select (if seen) or search for Live Server in the extension list
  3. Press the install button
  4. After the install, check a blue bottom status line right corner for the button text "Go Live"
  5. To use Go Live
    • Open an html file
    • Make any adjustments to your page
    • Save the changes
    • Press "Go Live"
