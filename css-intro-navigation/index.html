Create navigational links within a website

There are several techniques to navigate within a website. The semantic tag for navigation is the <nav> tag. YOu can navigate through your website by using realitive addressing. For our examples, we will use the unordered list tag (<ul>).

  • Page to page within the same folder
  • Page to page between folder
  • Staying on the current page
  • Going to a specific place on the current page (On page navigation)

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag, like other semantic tags, should have an <hx> tag. HOWEVER, this header is optional. If you omit the header tag you will receive a document outliner error: Untitled Section. Untill you are shown how to avoid this error message, it will be acceptable on document outline
